# Vertimpact Design Preview

Static design preview for the Vertimpact website concept.

This repository is prepared for client review and presentation via GitHub Pages.

## Overview

- Landing page concept for Vertimpact Ltd
- Current focus: header and hero section
- Built with plain HTML and CSS

## Preview

After GitHub Pages is enabled, the site will be available at:

`https://asvirkul.github.io/vertimpact-landing-preview/`

## Local Run

Open `index.html` in the browser or run the project with any simple local static server.

## Project Structure

- `index.html` — page markup
- `styles/` — base, component, and section styles
- `assets/` — fonts and images

## Notes

- This repository is intended as a visual review build
- Content and sections may continue evolving after client feedback
